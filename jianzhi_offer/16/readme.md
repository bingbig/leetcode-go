# 面试题16. [数值的整数次方](https://leetcode-cn.com/problems/shu-zhi-de-zheng-shu-ci-fang-lcof/)

实现函数`double Power(double base, int exponent)`，求`base`的`exponent`次方。不得使用库函数，同时不需要考虑大数问题。

示例 1:
```
输入: 2.00000, 10
输出: 1024.00000
```

示例 2:
```
输入: 2.10000, 3
输出: 9.26100
```

示例 3:
```
输入: 2.00000, -2
输出: 0.25000
```

解释: 2-2 = 1/22 = 1/4 = 0.25
